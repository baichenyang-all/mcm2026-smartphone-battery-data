# iPhone Task2 Data & Scripts

This folder contains the Task 2 scenario-level dataset and scripts produced by a teammate (iPhone-based workflow).

- `data_iphone_task2/raw/task2_real.csv`: scenario-level table (reference/measured TTE values) used in Task 2.
- `data_iphone_task2/processed/task2_results.csv`: computed results table.
- `scripts_iphone_task2/task2_validate.py` and `scripts_iphone_task2/task2_complete.py`: scripts generating Task 2 tables and figures.
- `data_iphone_task2/figures/`: figures produced by the scripts.

Note: This is scenario-level summary data (not raw device logs).
